# CMMC_SSP_Builder
A web app for building your CMMC Level 2 System Security Plan (SSP)
Key Features:

Client/System Management: Enter a client name and use Save/Load to work on multiple clients without losing progress (uses browser storage)
Organized Sections: Walks through all major SSP components:

System Identification
System Description & Purpose
System Environment & Architecture
Boundaries & Scope
Data Flow
Users & Access
Interconnections
Security Controls (all major families: AC, AU, CM, IA, IR, MA, MP, PE, RA, CA, SC, SI)
Continuous Monitoring
Additional Information

Helpful Prompts: Each field has guidance on what to include
Easy Output:

Click "Generate SSP Output" to see formatted text
"Copy to Clipboard" for pasting into Apptega
"Download as Text" to save a timestamped file

Workflow:

Enter your system name in the top field
Fill in the sections (you can skip any that don't apply)
Click "Save Progress" periodically
When done, "Generate SSP Output"
